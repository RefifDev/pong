# Pong Python 

This project was created by two friends to learn the basics of game development using the Pygame library. 

The code was developed for fun and as a learning exercise. We utilized AI assistance throughout the process to help us understand programming logic and solve technical challenges.

## Features
* **Single Player Mode**: A survival mode using a horizontal paddle at the bottom of the screen.
* **Two Player Mode**: A classic duel with vertical paddles on the left and right sides.
* **Difficulty Levels**: Four settings (Easy, Medium, Hard, Very Hard) that adjust the ball's speed.
* **Multi-language Support**: The interface is available in English, French, Italian, and Spanish.

## Controls
* **Single Player**: Use Left and Right Arrow keys to move the paddle.
* **Two Players**:
    * Player 1: W and S keys.
    * Player 2: Up and Down Arrow keys.
* **System**: 
    * Space: Pause or resume the game.
    * Escape: Quit the current session.

## Installation
This project requires Python and the Pygame library.

pip install pygame
python PONG.py
